## React Native Notes Application

A simple and clean Notes Application built with React Native (Expo + TypeScript).
This app allows users to register, log in, and manage their notes efficiently.

**Features**

- User Aunthentication(Register & Login
- Create Notes
- Edit Notes
- Categorize Notes
- View Notes List
- User Profile Screen
- Clean and responsive UI

## Tech Stack

- React Native
- Expo
- Typescript
- React Navigation

## Installation & Setup

git clone https://github.com/Refilwe15/react-native-notes-application-codetribe.git

cd react-native-notes-application

npm install

npx expo start
